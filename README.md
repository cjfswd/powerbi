# Unimed Nova Iguaçu / Camperj Dashboard - Faturamento 2025

Dashboard de faturamento para Unimed Nova Iguaçu e Camperj no modelo de atendimento domiciliar.

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui
- Recharts

## Executar

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
