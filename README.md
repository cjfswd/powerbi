# HealthMais Dashboard - Faturamento 2025

Dashboard de faturamento para Healthcare / HealthMais no modelo de atendimento domiciliar (Home Care).

## Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui
- Recharts

## Executar

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
