## Scripts manuais de teste

Estes arquivos sao testes e reproducoes locais que nao entram no `npm test`.

- `test_handler.ts`: smoke test direto do handler `/api/sheets`
- `test_sheets.ts`: leitura bruta da aba `REF_Procedimentos`
- `test_t_param.ts`: reproducao da URL com parametro `_t`

Use os atalhos do `package.json` para executa-los.
